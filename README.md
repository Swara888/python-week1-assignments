# Python Programming Course – Week 1 Assignments

This repository contains the **Week 1 assignments and practice programs** for the Python Programming Course by AXCENTRA.

## Topics Covered
- Python Syntax
- Variables and Data Types
- Conditional Statements (if-else)
- Loops
- Functions
- String and List Manipulation

---

## Assignments

### 1. Temperature Converter
A Python program that converts:
- Celsius → Fahrenheit
- Fahrenheit → Celsius

Uses functions to perform temperature conversions.

---

### 2. Student Grade Calculator
This program:
- Takes marks for three subjects as input
- Calculates the average
- Assigns a grade based on the average marks.

---

### 3. Even/Odd and Prime Number Checker
A program that:
- Checks whether a number is **even or odd**
- Determines whether the number is **prime or not**

---

## Mini Project

### Simple ATM Simulator
A basic ATM simulation program with:
- PIN-based login
- Balance checking
- Deposit functionality
- Withdrawal functionality

Concepts used:
- Loops
- Conditional statements
- Functions
- Basic user interaction
